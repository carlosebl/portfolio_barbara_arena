Precificação

Simplificação de um backoffice de alta complexidade de regras de negócio.

Problema

O principal problema do backoffice de preço era a alta complexidade de regras de negócio envolvidas no sistema. Assim, a descoberta conduzida teve como foco a simplificação da interface e do fluxo para o usuário. Para tal, realizei os seguintes processos:

Processos

Mapeamento dos fluxos existentes

Para análise dos pontos de possíveis melhorias, bem como das regras de negócio envolvidas na experiência.

Entrevistas com cooperativas

Para compreender as principais dores e ganhos dos usuários com a experiência em vigência.

User Flow

Para mapeamento dos passos e etapas necessários no sistema.

CSD

Para alinhamento de conhecimento do time, redução de incertezas e priorização de ideias.

Crazy 8

Para materialização das ideias priorizadas de forma co-criativa.

Prototipação

Prototipação interativa de alta fidelidade para teste com usuários.

Teste de usabilidade

Testes com usuários para análise do desempenho do protótipo criado.

Resultados

Através desses processos, foi possível compreender as principais dores dos usuários e pontos de melhoria do sistema. Foi proposta, testada e implementada uma nova interface com um fluxo de navegação e interação mais simplificado, com melhorias que foram acompanhadas pelo time de designops.

*As telas e boards dos processos desse projeto são protegidos por termo de confidencialidade.
